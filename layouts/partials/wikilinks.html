{{- /* Wikilinks para el digital garden. Recibe la Page; devuelve el HTML de .Content con los [[ ]] expandidos. Se opera sobre el HTML YA renderizado (.Content), no sobre .RawContent, para no romper shortcodes (listening/callout/ascii), TOC ni mermaid: los [[ ]] sobreviven como texto literal en el HTML final. Sintaxis: [[clave]] -> enlace con el título de la nota destino como texto [[clave|alias]] -> enlace con texto personalizado La "clave" es el último segmento del RelPermalink de la nota (p. ej. /posts/vigilancia-masiva/ -> "vigilancia-masiva"; /posts/codice/codice-00-nota-de-archivo/ -> "codice-00-nota-de-archivo"). Un [[ ]] sin nota coincidente queda marcado como enlace roto. IMPORTANTE — output formats: al renderizar en un formato distinto de "html" (p. ej. el formato "print" de la vista de impresión / export PDF), Hugo hace que .RelPermalink de las OTRAS páginas resuelva a ESE mismo formato. Sin corregirlo, la clave de todas las notas pasaba a ser "print.html" y TODOS los wikilinks aparecían rotos en la salida de impresión. Por eso la clave y el destino se derivan siempre del formato canónico "html". */ -}} {{- $page := . -}} {{- $content := $page.Content -}} {{- if strings.Contains $content "[[" -}} {{- range $page.Site.RegularPages -}} {{- /* permalink canónico (formato "html"), no el del formato en curso */ -}} {{- $url := .RelPermalink -}} {{- with .OutputFormats.Get "html" -}} {{- $url = .RelPermalink -}} {{- end -}} {{- $key := path.Base (strings.TrimRight "/" $url) -}} {{- $title := .Title | replaceRE `\$` `$$$$` -}} {{- /* alias primero: [[clave|alias]] */ -}} {{- $reAlias := printf `\[\[%s\|([^\]]+)\]\]` $key -}} {{- $content = replaceRE $reAlias (printf `$1` $url) $content -}} {{- /* forma simple: [[clave]] -> título de la nota */ -}} {{- $rePlain := printf `\[\[%s\]\]` $key -}} {{- $content = replaceRE $rePlain (printf `%s` $url $title) $content -}} {{- end -}} {{- /* lo que quede sin resolver = enlace roto */ -}} {{- $content = replaceRE `\[\[([^\]|]+)\|([^\]]+)\]\]` `$2` $content -}} {{- $content = replaceRE `\[\[([^\]|]+)\]\]` `$1` $content -}} {{- end -}} {{- return $content -}}
