Este es el sitio de ejemplo de **Vulpine Marrow**, un tema para Hugo: violeta
profundo sobre negro, tipografía Fira Code y estética terminal-clean, con modo
claro/oscuro, digital garden y diagramas mermaid.

El código y la documentación del tema están en
[github.com/Fennek115/vulpine-marrow](https://github.com/Fennek115/vulpine-marrow).
